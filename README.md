# Capstone : Finalisation de l'application "Réservation de salles"

# Objectifs de ce TP
 - Finalisation de l'application de réservation de salles
 - Création des jeux de données réalistes
 - Développement des scénarios de test complets
 - Implémentation d'un script de migration de base de données
 - Réalisation d'un rapport de performance

# Concepts clé :
 - Modèle de données complet avec relations entre entités et optimistic locking
 - Cache de second niveau pour améliorer les performances
 - Jeu de données réaliste pour tester l'application
 - Scénarios de test couvrant les fonctionnalités principales
 - Script de migration pour mettre à jour une base de données existante
 - Rapport de performance pour analyser et optimiser les performances

> # Technologies utilisées :
 - JDK 11+
 - Maven installé
 - MySQL
 - IDE -> IntelliJ IDEA
